The following changes were made in this release:

0.2.3.0
- Fixed the default configuration for arranging enums into an "Enumerations" region.
- Fixed an issue with the insertion of region elements.
- Fixed parsing of VB method/Handles identifiers beginning with an underscore.
- Preserve trailing comments for fields.
- Allow the configuration editor to load configuration files with invalid element references.
- Fixed handling of extra parentheses and improved validation for condition expressions.
